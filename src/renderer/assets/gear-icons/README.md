GearSpark — папка иконок для снаряжения

Здесь хранятся SVG-иконки для отображения снаряжения в профиле.

Как использовать:
- Поместите SVG-файлы сюда: `src/renderer/assets/gear-icons/your-icon.svg`
- В компоненте импортируйте по относительному пути или подключите динамически.

Примечание: броское имя папки — `gear-spark` (варианты: `gear-spark`, `gear-pulse`, `rideflare`).
