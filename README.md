#  ZAPFIT — Твой тренировочный хаб под контролем <img width="148" height="158" alt="изображение" src="https://github.com/user-attachments/assets/883eea88-2b93-4a38-8150-07d14d49df10" />


ZAPFIT — это  фитнес-трекер : анализирует твои тренировки, считает нагрузку (TSS), отслеживает гидратацию, сон, HRV и даже сравнивает заезды между собой. Всё в одном месте — красиво, быстро, с неоновым стилем.

---

## 🔥 Особенности

-  **AI Анализ** — баланс CTL/ATL/TSB, рекомендации по восстановлению
-  **Гидратация** — умный трекер воды с рекомендациями
- ️ **Календарь тренировок** — визуализация активности по дням
-  **Сравнение заездов** — пульс, мощность, время, дистанция — всё рядом
- 😴 **Здоровье** — сон, вес, BMI, HRV — полный контроль над телом
- 📈 **Динамика формы** — график прогресса за 60 дней
-  **GPX импорт + ручной ввод** — гибкость для любого типа активности

## 🚀 Установка и запуск

```bash
git clone https://github.com/твой_ник/zapfit.git
cd zapfit
npm install && npm start
# или
pip install -r requirements.txt && python app.py
```
